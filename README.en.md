## What Is Mobius?

Mobius is a self-evolving open-source Agent OS and AI workbench. It is designed to help users organize projects, teams, agents, models, knowledge, devices, compute resources, and extension apps into one traceable productivity system.

Instead of being a fixed tool, Mobius grows through real usage. A user can ask XiaoMo to create projects, split tasks, launch agent sessions, inspect progress, modify plugins, summarize results, and turn repeated workflows into reusable capabilities.

For the full introduction, please read the Chinese README in this repository or the original Mobius repository.
